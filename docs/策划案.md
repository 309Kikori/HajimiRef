# Hajimi Ref (哈基米Ref) 产品策划案

## 1. 产品愿景
打造一款**极简、高性能、便携**的参考图查看工具（Reference Image Viewer）。
灵感源自中文互联网meme：哈基米、叮咚鸡、圆头耄耋。
目标是成为美术人员、设计师、建模师在 Windows 平台上的轻量级替代方案，解决同类软件（如 PureRef）收费或体积过大、依赖复杂的问题。

## 2. 核心设计原则
*   **Simplicity (极简)**: 界面无多余按钮，最大化图片展示区域。
*   **Performance (高性能)**: 即使加载大量高清图片，缩放和平移也必须丝般顺滑。
*   **Portability (便携)**: 单个 EXE 文件，无需安装，即拷即用。

## 3. 目标用户
*   游戏美术设计师（原画、3D建模）
*   平面设计师
*   插画师
*   需要整理灵感素材的创意工作者

## 4. 功能规划

### 4.1 基础功能 (MVP - 已完成)
*   **无限画布**: 支持在二维平面上自由摆放图片。
*   **图片导入**: 
    *   支持拖拽文件（待完善）。
    *   支持右键菜单添加。
    *   支持 Ctrl+V 粘贴（支持图片数据和文件路径）。
*   **视图操作**:
    *   平移：中键拖拽 / 空格+左键。
    *   缩放：鼠标滚轮（以鼠标为中心）。
*   **图片操作**:
    *   移动：左键拖拽。
    *   缩放：Ctrl + 滚轮。
    *   删除：Delete 键。
    *   层级管理：点击图片自动置顶。
*   **文件管理**:
    *   保存看板为 `.sref` 文件（内嵌图片数据）。
    *   读取看板。
*   **系统集成**:
    *   始终置顶 (Always on Top) 模式。
    *   多语言支持 (中文/英文)。

### 4.2 进阶功能 (待开发)
*   **排列整理**: 自动排列图片（Pack images）。
*   **裁剪**: 在软件内对参考图进行简单裁剪。
*   **备注**: 添加文本便签。
*   **透明度**: 调节窗口整体透明度（Ghost Mode），方便覆盖在绘画软件上方描图。
*   **灰度模式**: 一键将所有图片转为灰度，方便观察素描关系。

## 5. 交互设计 (快捷键方案)
| 动作 | 快捷键 / 操作 |
| :--- | :--- |
| **移动画布** | 中键按住拖拽 / 空格 + 左键拖拽 |
| **缩放画布** | 鼠标滚轮 |
| **移动图片** | 左键按住图片拖拽 |
| **缩放图片** | Ctrl + 鼠标滚轮 |
| **删除图片** | Delete |
| **添加图片** | Ctrl + V / 右键菜单 |
| **菜单** | 右键点击空白处 |

## 6. 竞品分析
*   **PureRef**: 行业标杆，功能极其强大。缺点：新版收费，旧版下载困难。
*   **Kuadro**:以此为灵感，但已停止维护。
*   **Windows 照片查看器**: 只能看单图，无法拼贴。

**Hajimi Ref 的优势**: 完全免费、开源、Python 易于修改定制、单文件体积小。
