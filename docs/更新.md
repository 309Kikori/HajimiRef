# Hajimi Ref 更新说明

## 2026-01-29 周三 - 撤销重做系统与层级菜单

### 上午 11:57 - 修复：撤销/重做快捷键冲突问题
- **问题描述**：快捷键在画布上不生效，仅顶部菜单栏可用
- **根本原因**：`QShortcut` 和菜单栏 `QAction` 同时绑定了相同的 `Ctrl+Z`/`Ctrl+Shift+Z` 快捷键，导致 Qt 报出 "Ambiguous shortcut overload" 错误
- **解决方案**：
  - 移除 `MainWindow.py` 中重复的 `QShortcut` 绑定
  - 保留菜单栏的 `QAction` 快捷键设置
  - 设置 `QAction` 的 `ShortcutContext` 为 `ApplicationShortcut`，确保在任何控件获取焦点时都能触发
- **影响文件**：`HajimiRef_win11/Views/MainWindow.py`

### 上午 11:35 - 功能：添加图片层级菜单及子功能
- **新增功能**：右键点击图片时显示层级菜单（「层级」子菜单）
- **子菜单项**：
  - 「上移一个层级」- 将当前图片在图层顺序中向上移动一位
  - 「下移一个层级」- 将当前图片在图层顺序中向下移动一位
  - 「移动至最上层」- 将当前图片移动到所有图层最上方
  - 「移动至最下层」- 将当前图片移动到所有图层最下方
- **快捷键绑定**：
  - `Ctrl+Up` / `Ctrl+Down` - 上移/下移
  - `Ctrl+Shift+Up` / `Ctrl+Shift+Down` - 移至最上/最下
- **技术实现**：Windows 和 macOS 双端同步支持，均集成撤销/重做功能

### 上午 10:29 - 功能：完整撤销/重做系统
- **功能描述**：为整个应用添加完整的撤销/重做（Undo/Redo）系统
- **技术实现**：
  - **Windows**：新增 `HajimiRef_win11/Models/UndoManager.py`，实现基于命令模式的撤销管理器
  - **macOS**：新增 `HajimiRef_macos/Models/UndoManager.swift`，集成 macOS 原生撤销机制
  - 支持的操作：移动图片、缩放图片、旋转图片、删除图片、粘贴图片、调整图层顺序等
- **UI 集成**：
  - 顶部菜单栏添加「编辑」菜单，包含撤销/重做选项
  - 快捷键：`Ctrl+Z`（撤销）、`Ctrl+Shift+Z`/`Ctrl+Y`（重做）
- **双语支持**：中英文本地化字符串完整覆盖

### 上午 10:02 - 功能：自动重置画板功能
- **新增功能**：添加定时器自动重置画板和用户控制选项
- **技术实现**：

## 2026-01-28 周二 - 性能优化与体验提升

### 下午 18:00 - 优化：固定扩展画板边界机制
- **改动内容**：重构画板边界实现，从动态扩展改为固定扩展模式
- **技术细节**：
  - 设置合理的扩展边距（buffer），避免图片越界
  - 优化边界检测逻辑，提升大尺寸图片的处理效率

### 下午 17:38 - 性能优化：渲染区域优化
- **优化内容**：
  - 引入「活动区域」（Active Area）概念，仅渲染可见范围内的内容
  - 添加渲染边距（Padding），预渲染视口边缘区域以确保流畅滚动
- **效果**：大幅提升大尺寸画板的渲染性能，降低 CPU/GPU 占用

### 下午 17:23 - 体验优化：移除复制成功提示
- **改动内容**：复制图片到剪贴板后不再显示成功提示弹窗
- **优化理由**：减少不必要的弹窗干扰，提升用户操作流畅度
- **技术实现**：移除 `Canvas.py` 中的复制成功消息框调用

## A1.0.2 (2025-11-28) - 性能与体验大修
### 新增功能
*   **始终置顶**: 在设置菜单中增加了"始终置顶 (Always on Top)"选项，方便在绘画时参考。
*   **增强粘贴**: 优化了 `Ctrl+V` 逻辑，现在不仅支持粘贴截图，还支持从资源管理器直接复制文件后粘贴。

### 优化
*   **视口裁剪 (Viewport Clipping)**: 重写了图像渲染管线。现在只渲染屏幕可见区域的图像切片。
    *   *效果*: 彻底解决了图片放大倍数过大时（如 >500%）导致的严重卡顿和内存溢出问题。
*   **动态重采样 (Dynamic Resampling)**: 
    *   滚动/拖拽时使用 `Nearest Neighbor` 插值，保证 60fps 流畅度。
    *   停止操作 200ms 后自动切换为 `Lanczos/Bilinear` 插值，保证画质清晰。
*   **汉化补全**: 修复了界面中遗漏的英文文本（如 Exit, Always on Top）。
